Fonctionnement :
1. Le QR ouvre une adresse comme index.html?code=22G6-DPV-RH
2. La page récupère automatiquement le code
3. L’utilisateur saisit le mot de passe unique
4. Le système affiche le code ou le valide

Important :
pour que le scan ouvre vraiment cette page sur un téléphone, il faut héberger le fichier en ligne puis générer les QR avec ton vrai lien.

Guide de mise en place

Étape 1 : héberge ce fichier index.html.

Étape 2 : remplace dans le fichier qr_targets_template.csv le texte https://TON-LIEN par ton vrai lien.

Étape 3 : génère ensuite les QR à partir de la colonne url_qr_a_encoder.

Étape 4 : à chaque scan, la page récupère le code automatiquement et demande le mot de passe.
